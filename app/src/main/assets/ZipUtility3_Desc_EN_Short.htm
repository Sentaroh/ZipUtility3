1.    Capabbility

It is an application for creating, editing, and extracting ZIP file.

      You can create and extraction and browsing of password protected Zip file.

      Save and editing the ZIP file on the MicroSD/USB Flash

      Support for the Japanese file name(Automatic identification or manual)

      You can specify the file extensions to be stored without compression (the default is jpg, mpg, mp4, mp4, pdf, etc.)

      In conjunction with external applications, other popular file formats also view

      You can copy, cut between the local folder and the ZIP folder

      Search the file name(In the results list to display the directory when you tap the file name)

 

How to use, you can either tap the ZIP file with a tiny file manager, please select the ZipUtility3 from the app to open tap the ZIP file in the other file manager.
