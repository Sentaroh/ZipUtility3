1.    Capabbility

It is an application for creating, editing, and extracting ZIP file.

      You can create and extraction and browsing of password protected Zip file.

      Save and editing the ZIP file on the MicroSD/USB Flash

      Support for the Japanese file name(Automatic identification or manual)

      You can specify the file extensions to be stored without compression (the default is jpg, mpg, mp4, mp4, pdf, etc.)

      In conjunction with external applications, other popular file formats also view

      Simple File Manager

      You can copy and cut files in the local folder(*1)

      You can copy and cut between local and zip folders

      File name search in local and zip folders

*1 Local storage indicates internal storage, MicroSD and USB Flash.

 

How to use, you can either tap the ZIP file with a tiny file manager, please select the ZipUtility3 from the app to open tap the ZIP file in the other file manager.

2.    Screen Description

Select an item and tap the icon to operate on the directory/file. Tap and hold the icon to display an explanation.

2.1 Description of the local folder icon

 Creating a Directory or File

 Share the selected item.

 Rename the selected item.

 Paste a copy or cut item (the cut item will be deleted when the pasting is complete)

 Copy the selected items.

 Cut the selected items.

 Compress the selected items.

 Removes the selected item.

 Select All.

 Deselect everything.

 

2.2 Description of the ZIP folder icon

 Create a ZIP file

 Open the ZIP file

 Paste a copy or clipped item (the clipped item will be deleted when the pasting is complete)

 Copy the selected items.

 Cut the selected items.

 Extract the selected items

 Removes the selected item.

 Select All.

 Deselect everything.
